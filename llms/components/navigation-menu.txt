# Navigation Menu Component

A collection of links for navigating websites with support for dropdown menus.

---

## Overview

The NavigationMenu component provides a horizontal navigation bar commonly used for site-wide navigation. It supports dropdown menus, links, and visual indicators for the active menu item.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.NavigationMenu
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/NavigationMenu

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.NavigationMenu
```

## API Reference

### Main Component: `<NavigationMenu>`

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `EnableKeyboardNavigation` | `bool` | `false` | Enables arrow key navigation within dropdown menus |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Navigation content |

**Behavior:**
- Menu automatically closes when navigation occurs (listens to `LocationChanged`)
- Dropdowns open on hover and close when mouse leaves
- When `EnableKeyboardNavigation="true"`, arrow keys navigate between items in dropdowns

### Sub-components

#### `<NavigationMenuList>`

Container for menu items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Menu items |

#### `<NavigationMenuItem>`

Individual menu item container.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Trigger and content |

#### `<NavigationMenuTrigger>`

Button that opens a dropdown menu.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Trigger text/content |

#### `<NavigationMenuContent>`

Dropdown content panel.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Dropdown content |

#### `<NavigationMenuLink>`

Navigation link.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Href` | `string?` | `null` | Link destination |
| `Active` | `bool` | `false` | Whether link is active |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Link content |

#### `<NavigationMenuIndicator>`

Visual indicator for active menu item.

#### `<NavigationMenuViewport>`

Container for dropdown content with animations.

## Basic Usage

```razor
<NavigationMenu>
    <NavigationMenuList>
        <NavigationMenuItem>
            <NavigationMenuLink Href="/">Home</NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
            <NavigationMenuLink Href="/about">About</NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
            <NavigationMenuLink Href="/contact">Contact</NavigationMenuLink>
        </NavigationMenuItem>
    </NavigationMenuList>
</NavigationMenu>
```

## Keyboard Navigation

Keyboard navigation within dropdown content is **opt-in** via the `EnableKeyboardNavigation` parameter.

### Enabling Keyboard Navigation

```razor
<NavigationMenu EnableKeyboardNavigation="true">
    <NavigationMenuList>
        <!-- Menu items -->
    </NavigationMenuList>
</NavigationMenu>
```

### At Trigger Level (always available)

| Key | Action |
|-----|--------|
| `ArrowRight` | Move focus to next trigger |
| `ArrowLeft` | Move focus to previous trigger |
| `ArrowDown` | Open dropdown and focus first item |
| `Enter` / `Space` | Toggle dropdown open/closed |
| `Escape` | Close dropdown |

### Within Dropdown Content (requires `EnableKeyboardNavigation="true"`)

| Key | Action |
|-----|--------|
| `ArrowDown` | Move focus to next link |
| `ArrowUp` | Move focus to previous link |
| `Home` | Jump to first link |
| `End` | Jump to last link |
| `Enter` | Follow focused link |
| `Escape` | Close dropdown, return focus to trigger |

### Focus Behavior

- Dropdown opens on hover (mouse) or `ArrowDown`/`Enter` (keyboard)
- When keyboard navigation is enabled, focus ring appears around focused items
- Focus wraps from last item to first (and vice versa)
- Closing dropdown returns focus to the trigger button
- Menu automatically closes when a link is clicked and navigation occurs

## Accessibility

- Full keyboard navigation support (see above)
- Proper ARIA attributes (`role="menu"`, `aria-haspopup`, `aria-expanded`)
- Focus management within dropdowns
- Visual focus indicators for keyboard users

## Common Patterns

### Header Navigation

```razor
<header class="border-b">
    <div class="container flex items-center justify-between py-4">
        <div class="font-bold text-xl">Logo</div>
        <NavigationMenu>
            <NavigationMenuList>
                <!-- Navigation items -->
            </NavigationMenuList>
        </NavigationMenu>
        <Button>Sign In</Button>
    </div>
</header>
```

### Responsive Navigation

For mobile responsiveness, consider combining with Sheet component for a mobile menu:

```razor
@if (isMobile)
{
    <Sheet>
        <!-- Mobile menu -->
    </Sheet>
}
else
{
    <NavigationMenu>
        <!-- Desktop navigation -->
    </NavigationMenu>
}
```

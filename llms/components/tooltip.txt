# Tooltip Component

Brief informational popup on hover or focus with intelligent positioning.

---

## Overview

The Tooltip component displays brief information on hover or focus. Features delay control, intelligent positioning, and accessibility support.

**Features:**
- Hover and focus triggers
- Configurable delays
- Intelligent positioning
- Portal rendering
- ARIA attributes
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Tooltip
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Tooltip

**Note:** Requires `<PortalHost />` in your layout.

---

## API Reference

### Components

- `<TooltipProvider>` - Context provider (optional)
- `<Tooltip>` - Container
- `<TooltipTrigger>` - Trigger element
- `<TooltipContent>` - Tooltip content

---

## Basic Usage

```razor
<Tooltip>
    <TooltipTrigger AsChild>
        <Button Size="ButtonSize.Icon">
            <LucideIcon Name="help-circle" Size="16" />
        </Button>
    </TooltipTrigger>
    <TooltipContent>
        <p>Help information</p>
    </TooltipContent>
</Tooltip>
```

---

## Examples

### Icon with Tooltip

```razor
<Tooltip>
    <TooltipTrigger AsChild>
        <Button Size="ButtonSize.Icon" Variant="ButtonVariant.Ghost">
            <LucideIcon Name="settings" Size="16" />
        </Button>
    </TooltipTrigger>
    <TooltipContent>
        Settings
    </TooltipContent>
</Tooltip>
```

### Text with Tooltip

```razor
<Tooltip>
    <TooltipTrigger AsChild>
        <span class="underline cursor-help">Hover me</span>
    </TooltipTrigger>
    <TooltipContent>
        Additional information about this text
    </TooltipContent>
</Tooltip>
```

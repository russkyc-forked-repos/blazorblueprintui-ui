# Input Group Component

Enhanced inputs with icons, buttons, and addons for complex input compositions.

---

## Overview

The InputGroup component enhances standard inputs by adding icons, buttons, and text addons before or after the input field. Perfect for search boxes, URL inputs, and actions-enabled fields.

**Features:**
- Icon addons (InputGroupIcon)
- Button addons (InputGroupButton)
- Text addons (InputGroupText)
- Start and end positioning
- Seamless integration with Input component
- Dark mode compatible

## API Reference

### `<InputGroup>`

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Input and addons |

### Sub-components

- `<InputGroupIcon>` - Icon addon
- `<InputGroupButton>` - Button addon
- `<InputGroupText>` - Text addon

---

## Basic Usage

```razor
<InputGroup>
    <InputGroupIcon>
        <LucideIcon Name="search" Size="16" />
    </InputGroupIcon>
    <Input Placeholder="Search..." />
</InputGroup>
```

## Examples

### Search with Icon

```razor
<InputGroup>
    <InputGroupIcon>
        <LucideIcon Name="search" Size="16" />
    </InputGroupIcon>
    <Input
        Type="InputType.Search"
        @bind-Value="searchTerm"
        Placeholder="Search..." />
</InputGroup>

@code {
    private string searchTerm = "";
}
```

### Search with Clear Button

```razor
<InputGroup>
    <InputGroupIcon>
        <LucideIcon Name="search" Size="16" />
    </InputGroupIcon>
    <Input
        Type="InputType.Search"
        @bind-Value="query"
        Placeholder="Search..." />
    @if (!string.IsNullOrEmpty(query))
    {
        <InputGroupButton>
            <Button
                Variant="ButtonVariant.Ghost"
                Size="ButtonSize.Icon"
                OnClick="@(() => query = "")">
                <LucideIcon Name="x" Size="16" />
            </Button>
        </InputGroupButton>
    }
</InputGroup>

@code {
    private string query = "";
}
```

### Password with Toggle Visibility

```razor
<InputGroup>
    <InputGroupIcon>
        <LucideIcon Name="lock" Size="16" />
    </InputGroupIcon>
    <Input
        Type="@(showPassword ? InputType.Text : InputType.Password)"
        @bind-Value="password"
        Placeholder="Enter password" />
    <InputGroupButton>
        <Button
            Variant="ButtonVariant.Ghost"
            Size="ButtonSize.Icon"
            OnClick="@(() => showPassword = !showPassword)">
            <LucideIcon Name="@(showPassword ? "eye-off" : "eye")" Size="16" />
        </Button>
    </InputGroupButton>
</InputGroup>

@code {
    private string password = "";
    private bool showPassword = false;
}
```

### Copy to Clipboard

```razor
<InputGroup>
    <Input Value="@shareUrl" Disabled="true" />
    <InputGroupButton>
        <Button OnClick="CopyToClipboard">
            <LucideIcon Name="copy" Size="16" />
            Copy
        </Button>
    </InputGroupButton>
</InputGroup>

@code {
    private string shareUrl = "https://example.com/share/abc123";

    private async Task CopyToClipboard()
    {
        await JS.InvokeVoidAsync("navigator.clipboard.writeText", shareUrl);
    }
}
```

## Common Patterns

### Command Search

```razor
<InputGroup>
    <InputGroupIcon>
        <LucideIcon Name="search" Size="16" />
    </InputGroupIcon>
    <Input
        @bind-Value="command"
        Placeholder="Type a command..."
        @onkeydown="HandleKeyDown" />
    <InputGroupText Class="text-xs text-muted-foreground">
        <kbd>âŒ˜K</kbd>
    </InputGroupText>
</InputGroup>

@code {
    private string command = "";

    private void HandleKeyDown(KeyboardEventArgs e)
    {
        if (e.Key == "Enter")
        {
            // Execute command
        }
    }
}
```

```razor
<InputGroup>
    <Input Type="InputType.File" />
    <InputGroupButton>
        <Button Variant="ButtonVariant.Secondary">
            Browse
        </Button>
    </InputGroupButton>
</InputGroup>
```

# Badge Component

Label component for displaying status, categories, and metadata with multiple variants.

---

## Overview

The Badge component displays small labels for status indicators, categories, tags, and metadata. Features multiple visual variants and customizable styling.

**Features:**
- 4 variants (default, secondary, destructive, outline)
- Customizable styling
- Icon support
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Badge
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Badge

---

## API Reference

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Variant` | `BadgeVariant` | `Default` | Visual style (Default, Secondary, Destructive, Outline) |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Badge content |

---

## Basic Usage

```razor
<Badge>Default</Badge>
<Badge Variant="BadgeVariant.Secondary">Secondary</Badge>
<Badge Variant="BadgeVariant.Destructive">Destructive</Badge>
<Badge Variant="BadgeVariant.Outline">Outline</Badge>
```

---

## Examples

### Status Badges

```razor
<Badge Variant="BadgeVariant.Default">Active</Badge>
<Badge Variant="BadgeVariant.Secondary">Pending</Badge>
<Badge Variant="BadgeVariant.Destructive">Failed</Badge>
<Badge Variant="BadgeVariant.Outline">Inactive</Badge>
```

### With Icons

```razor
<Badge>
    <LucideIcon Name="check" Size="12" Class="mr-1" />
    Verified
</Badge>

<Badge Variant="BadgeVariant.Destructive">
    <LucideIcon Name="x" Size="12" Class="mr-1" />
    Error
</Badge>
```

### In Table

```razor
<table>
    <tr>
        <td>User 1</td>
        <td><Badge Variant="BadgeVariant.Default">Admin</Badge></td>
        <td><Badge>Active</Badge></td>
    </tr>
    <tr>
        <td>User 2</td>
        <td><Badge Variant="BadgeVariant.Secondary">User</Badge></td>
        <td><Badge Variant="BadgeVariant.Outline">Inactive</Badge></td>
    </tr>
</table>
```

### Tag List

```razor
<div class="flex flex-wrap gap-2">
    <Badge Variant="BadgeVariant.Outline">React</Badge>
    <Badge Variant="BadgeVariant.Outline">TypeScript</Badge>
    <Badge Variant="BadgeVariant.Outline">Tailwind CSS</Badge>
    <Badge Variant="BadgeVariant.Outline">Blazor</Badge>
</div>
```

### Notification Count

```razor
<Button Variant="ButtonVariant.Outline" Class="relative">
    <LucideIcon Name="bell" Size="16" />
    Notifications
    <Badge Class="absolute -top-2 -right-2 px-1.5 py-0.5">
        3
    </Badge>
</Button>
```

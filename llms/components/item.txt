# Item Component

Flexible list item component with media (avatar/icon), content, and action areas.

---

## Overview

The Item component provides a flexible list item structure with media (avatar, icon), content sections, and action areas. Perfect for building lists, menus, and navigation items.

**Features:**
- Media section (avatar, icon)
- Title and description sections
- Action area
- Flexible layout
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Item
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Item

---

## Basic Usage

```razor
<Item>
    <ItemMedia>
        <Avatar>
            <AvatarFallback>JD</AvatarFallback>
        </Avatar>
    </ItemMedia>
    <ItemContent>
        <ItemTitle>John Doe</ItemTitle>
        <ItemDescription>john@example.com</ItemDescription>
    </ItemContent>
    <ItemAction>
        <Button Size="ButtonSize.Icon" Variant="ButtonVariant.Ghost">
            <LucideIcon Name="more-horizontal" Size="16" />
        </Button>
    </ItemAction>
</Item>
```

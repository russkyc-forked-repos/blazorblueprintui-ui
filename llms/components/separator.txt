# Separator Component

Visual divider for content sections with horizontal and vertical orientation.

---

## Overview

The Separator component provides a visual divider to separate content sections. Supports horizontal and vertical orientations with customizable styling.

**Features:**
- Horizontal and vertical orientations
- Customizable styling
- ARIA role="separator"
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Separator
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Separator

---

## API Reference

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Orientation` | `Orientation` | `Horizontal` | Orientation (Horizontal or Vertical) |
| `Decorative` | `bool` | `false` | Whether separator is decorative only |
| `Class` | `string?` | `null` | Additional CSS classes |

---

## Basic Usage

```razor
<Separator />
```

---

## Examples

### Horizontal Separator

```razor
<div class="space-y-4">
    <p>Content above</p>
    <Separator />
    <p>Content below</p>
</div>
```

### Vertical Separator

```razor
<div class="flex h-5 items-center space-x-4">
    <span>Link 1</span>
    <Separator Orientation="Orientation.Vertical" />
    <span>Link 2</span>
    <Separator Orientation="Orientation.Vertical" />
    <span>Link 3</span>
</div>
```

### In Navigation

```razor
<div class="flex items-center space-x-2">
    <Button Variant="ButtonVariant.Ghost">Home</Button>
    <Separator Orientation="Orientation.Vertical" Class="h-6" />
    <Button Variant="ButtonVariant.Ghost">About</Button>
    <Separator Orientation="Orientation.Vertical" Class="h-6" />
    <Button Variant="ButtonVariant.Ghost">Contact</Button>
</div>
```

### In Card

```razor
<Card>
    <CardHeader>
        <CardTitle>Settings</CardTitle>
    </CardHeader>
    <CardContent Class="space-y-4">
        <div>
            <Label>Email Notifications</Label>
            <Switch @bind-Checked="emailNotif" />
        </div>

        <Separator />

        <div>
            <Label>Push Notifications</Label>
            <Switch @bind-Checked="pushNotif" />
        </div>
    </CardContent>
</Card>

@code {
    private bool emailNotif = true;
    private bool pushNotif = false;
}
```

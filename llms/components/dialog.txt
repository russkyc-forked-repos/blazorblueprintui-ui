# Dialog Component

Modal dialog with backdrop, focus management, and shadcn/ui design.

---

## Overview

The Dialog component provides accessible modal dialogs with backdrop overlay, focus trapping, escape key handling, and composition-based structure.

**Features:**
- Backdrop overlay
- Focus trap
- Escape key to close
- Controlled and uncontrolled modes
- Composition-based structure (header, content, footer)
- Portal rendering
- Scroll lock
- ARIA attributes
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Dialog
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Dialog

**Note:** Requires `<PortalHost />` in your layout for proper rendering.

---

## API Reference

### Main Components

- `<Dialog>` - Container
- `<DialogTrigger>` - Opens dialog
- `<DialogContent>` - Content container
- `<DialogHeader>` - Header section
- `<DialogTitle>` - Title
- `<DialogDescription>` - Description
- `<DialogFooter>` - Footer/actions
- `<DialogClose>` - Close button

---

## Basic Usage

```razor
<Dialog>
    <DialogTrigger AsChild>
        <Button>Open Dialog</Button>
    </DialogTrigger>
    <DialogContent>
        <DialogHeader>
            <DialogTitle>Dialog Title</DialogTitle>
            <DialogDescription>Dialog description</DialogDescription>
        </DialogHeader>
        <p>Dialog content goes here</p>
        <DialogFooter>
            <DialogClose AsChild>
                <Button Variant="ButtonVariant.Outline">Cancel</Button>
            </DialogClose>
            <Button>Confirm</Button>
        </DialogFooter>
    </DialogContent>
</Dialog>
```

# Command Component

Command palette for quick actions and navigation with keyboard shortcuts and search.

---

## Overview

The Command component provides a searchable command palette for quick access to actions, navigation, and features. Features keyboard navigation, grouping, and empty states.

**Features:**
- Search/filter commands
- Keyboard navigation
- Command groups
- Empty state
- Icons and shortcuts
- Dark mode compatible

## API Reference

### Components

- `<Command>` - Container
- `<CommandInput>` - Search input
- `<CommandList>` - Results list
- `<CommandEmpty>` - Empty state
- `<CommandGroup>` - Group container
- `<CommandItem>` - Command item
- `<CommandSeparator>` - Divider

---

## Basic Usage

```razor
<Command>
    <CommandInput Placeholder="Type a command..." />
    <CommandList>
        <CommandEmpty>No results found.</CommandEmpty>
        <CommandGroup Heading="Suggestions">
            <CommandItem OnSelect="@(() => Navigate("/"))">
                <LucideIcon Name="house" Size="16" />
                <span>Dashboard</span>
            </CommandItem>
            <CommandItem OnSelect="@(() => Navigate("/settings"))">
                <LucideIcon Name="settings" Size="16" />
                <span>Settings</span>
            </CommandItem>
        </CommandGroup>
    </CommandList>
</Command>

@code {
    private void Navigate(string url)
    {
        NavigationManager.NavigateTo(url);
    }
}
```

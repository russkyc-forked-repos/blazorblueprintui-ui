# Sheet Component

Side panel that slides in from viewport edges with overlay and animations.

---

## Overview

The Sheet component displays a side panel that slides in from the top, right, bottom, or left edge of the viewport. Features smooth animations, backdrop overlay, and focus management.

**Features:**
- 4 positions (top, right, bottom, left)
- Smooth slide animations
- Backdrop overlay
- Focus trap
- Escape key to close
- Portal rendering
- ARIA attributes
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Sheet
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Sheet

**Note:** Requires `<PortalHost />` in your layout.

---

## API Reference

### Main Components

- `<Sheet>` - Container
- `<SheetTrigger>` - Opens sheet
- `<SheetContent>` - Content panel
- `<SheetHeader>` - Header section
- `<SheetTitle>` - Title
- `<SheetDescription>` - Description
- `<SheetFooter>` - Footer section
- `<SheetClose>` - Close button

---

## Basic Usage

```razor
<Sheet>
    <SheetTrigger AsChild>
        <Button>Open Sheet</Button>
    </SheetTrigger>
    <SheetContent>
        <SheetHeader>
            <SheetTitle>Sheet Title</SheetTitle>
            <SheetDescription>Sheet description</SheetDescription>
        </SheetHeader>
        <p>Sheet content</p>
        <SheetFooter>
            <SheetClose AsChild>
                <Button>Close</Button>
            </SheetClose>
        </SheetFooter>
    </SheetContent>
</Sheet>
```

---

## Examples

### Sheet from Right (Default)

```razor
<Sheet>
    <SheetTrigger AsChild>
        <Button>Open Settings</Button>
    </SheetTrigger>
    <SheetContent Side="SheetSide.Right">
        <SheetHeader>
            <SheetTitle>Settings</SheetTitle>
        </SheetHeader>
        <!-- Settings content -->
    </SheetContent>
</Sheet>
```

### Sheet from Bottom

```razor
<Sheet>
    <SheetTrigger AsChild>
        <Button>Open Menu</Button>
    </SheetTrigger>
    <SheetContent Side="SheetSide.Bottom">
        <!-- Mobile menu content -->
    </SheetContent>
</Sheet>
```

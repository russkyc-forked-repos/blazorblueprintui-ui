# Menubar Component

Desktop application-style menu bar with dropdown menus.

---

## Overview

The Menubar component provides a horizontal menu bar commonly found in desktop applications. It supports dropdown menus with items, separators, labels, and keyboard shortcuts.

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.Menubar
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/Menubar

---

## Installation

```bash
dotnet add package BlazorUI.Components
```

Add to `_Imports.razor`:

```razor
@using BlazorUI.Components.Menubar
```

## API Reference

### Main Component: `<Menubar>`

Horizontal menu bar container.

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | MenubarMenu elements |

### Sub-components

#### `<MenubarMenu>`

Individual menu in the bar.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Trigger and content |

#### `<MenubarTrigger>`

Button that opens the dropdown.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Menu label |

#### `<MenubarContent>`

Dropdown menu container.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Align` | `MenubarContentAlign` | `Start` | Alignment (Start, Center, End) |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Menu items |

#### `<MenubarItem>`

Clickable menu item.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `OnClick` | `EventCallback` | - | Click handler |
| `Disabled` | `bool` | `false` | Disables the item |
| `Inset` | `bool` | `false` | Adds left padding |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Item content |

#### `<MenubarLabel>`

Non-interactive section label.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Inset` | `bool` | `false` | Adds left padding |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Label text |

#### `<MenubarSeparator>`

Visual divider between items.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |

#### `<MenubarShortcut>`

Keyboard shortcut hint.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Shortcut text |

#### `<MenubarCheckboxItem>`

A menu item with a checkbox that can be toggled.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Checked` | `bool` | `false` | Whether the checkbox is checked |
| `CheckedChanged` | `EventCallback<bool>` | - | Callback when checked state changes |
| `Disabled` | `bool` | `false` | Whether the item is disabled |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | - | Item content |

## Basic Usage

```razor
<Menubar>
    <MenubarMenu>
        <MenubarTrigger>File</MenubarTrigger>
        <MenubarContent>
            <MenubarItem OnClick="@(() => HandleAction("New Tab"))">
                New Tab
                <MenubarShortcut>Ctrl+T</MenubarShortcut>
            </MenubarItem>
            <MenubarItem OnClick="@(() => HandleAction("New Window"))">
                New Window
                <MenubarShortcut>Ctrl+N</MenubarShortcut>
            </MenubarItem>
            <MenubarSeparator />
            <MenubarItem OnClick="@(() => HandleAction("Print"))">
                Print...
                <MenubarShortcut>Ctrl+P</MenubarShortcut>
            </MenubarItem>
        </MenubarContent>
    </MenubarMenu>
    <MenubarMenu>
        <MenubarTrigger>Edit</MenubarTrigger>
        <MenubarContent>
            <MenubarItem>Undo <MenubarShortcut>Ctrl+Z</MenubarShortcut></MenubarItem>
            <MenubarItem>Redo <MenubarShortcut>Ctrl+Y</MenubarShortcut></MenubarItem>
            <MenubarSeparator />
            <MenubarItem>Cut <MenubarShortcut>Ctrl+X</MenubarShortcut></MenubarItem>
            <MenubarItem>Copy <MenubarShortcut>Ctrl+C</MenubarShortcut></MenubarItem>
            <MenubarItem>Paste <MenubarShortcut>Ctrl+V</MenubarShortcut></MenubarItem>
        </MenubarContent>
    </MenubarMenu>
</Menubar>

@code {
    private void HandleAction(string action)
    {
        Console.WriteLine($"Action: {action}");
    }
}
```

## Behavior

- Menus open on click and close when clicking outside
- Moving mouse to another trigger opens that menu
- Escape key closes the open menu
- Keyboard navigation between menu triggers

---

## Accessibility

- Uses proper menubar ARIA roles
- Keyboard navigation with arrow keys
- Menu items are focusable with Enter/Space activation
- Screen reader announcements for menu state

## Functional Keyboard Shortcuts

The `<MenubarShortcut>` component displays keyboard shortcuts visually. To make them actually work, use the `IKeyboardShortcutService`:

```razor
@inject IKeyboardShortcutService KeyboardShortcuts
@implements IAsyncDisposable

<Menubar>
    <MenubarMenu>
        <MenubarTrigger>File</MenubarTrigger>
        <MenubarContent>
            <MenubarItem OnClick="HandleSave">
                Save
                <MenubarShortcut>Ctrl+S</MenubarShortcut>
            </MenubarItem>
            <MenubarItem OnClick="HandleNew">
                New
                <MenubarShortcut>Ctrl+Shift+N</MenubarShortcut>
            </MenubarItem>
        </MenubarContent>
    </MenubarMenu>
</Menubar>

@code {
    private IDisposable? _shortcutSave;
    private IDisposable? _shortcutNew;

    protected override async Task OnAfterRenderAsync(bool firstRender)
    {
        if (firstRender)
        {
            _shortcutSave = await KeyboardShortcuts.RegisterAsync("Ctrl+S", HandleSaveAsync);
            _shortcutNew = await KeyboardShortcuts.RegisterAsync("Ctrl+Shift+N", HandleNewAsync);
        }
    }

    private void HandleSave() => _ = HandleSaveAsync();
    private void HandleNew() => _ = HandleNewAsync();
    private Task HandleSaveAsync() => Task.CompletedTask;
    private Task HandleNewAsync() => Task.CompletedTask;

    public ValueTask DisposeAsync()
    {
        _shortcutSave?.Dispose();
        _shortcutNew?.Dispose();
        return ValueTask.CompletedTask;
    }
}
```

**Note:** Use `Ctrl+Shift+` variants for shortcuts that conflict with browser defaults (Ctrl+N, Ctrl+T, Ctrl+W, Ctrl+P).

For full documentation, see: **llms/components/keyboard-shortcut-service.txt**

# Radio Group Component

Mutually exclusive selection control with keyboard navigation.

---

## Overview

The RadioGroup component provides a set of radio buttons where only one option can be selected at a time. Features keyboard navigation, two-way binding, and full accessibility.

**Features:**
- Mutually exclusive selection
- Two-way binding (@bind-Value)
- Keyboard navigation (Arrow keys)
- Disabled state (group and individual items)
- ARIA attributes
- Form validation integration
- Dark mode compatible

## API Reference

### `<RadioGroup>`

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `string?` | `null` | Selected value (supports @bind-Value) |
| `ValueChanged` | `EventCallback<string?>` | - | Callback when selection changes |
| `Disabled` | `bool` | `false` | Whether entire group is disabled |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Radio items |

### `<RadioGroupItem>`

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `string` | **Required** | Value for this radio button |
| `Disabled` | `bool` | `false` | Whether this item is disabled |
| `Id` | `string?` | `null` | HTML id for label association |

---

## Basic Usage

```razor
<RadioGroup @bind-Value="selectedOption">
    <RadioGroupItem Value="option1" Id="opt1" />
    <RadioGroupItem Value="option2" Id="opt2" />
    <RadioGroupItem Value="option3" Id="opt3" />
</RadioGroup>

@code {
    private string selectedOption = "option1";
}
```

### Entire Group Disabled


### Form Integration

```razor
<EditForm Model="@model" OnValidSubmit="@HandleSubmit">
    <DataAnnotationsValidator />

    <Field>
        <FieldLabel>Select Plan</FieldLabel>
        <FieldContent>
            <RadioGroup @bind-Value="model.Plan">
                <div class="flex items-center space-x-2">
                    <RadioGroupItem Value="free" Id="free" />
                    <Label For="free">Free</Label>
                </div>
                <div class="flex items-center space-x-2">
                    <RadioGroupItem Value="pro" Id="pro" />
                    <Label For="pro">Pro</Label>
                </div>
            </RadioGroup>
        </FieldContent>
        <FieldError>
            <ValidationMessage For="@(() => model.Plan)" />
        </FieldError>
    </Field>

    <Button Type="ButtonType.Submit">Continue</Button>
</EditForm>

@code {
    private PlanModel model = new();

    private void HandleSubmit() { }

    public class PlanModel
    {
        [Required]
        public string Plan { get; set; } = "";
    }
}
```

## Accessibility

### Keyboard Navigation
- **Arrow Up/Down**: Navigate between options
- **Arrow Left/Right**: Navigate between options
- **Space**: Select focused option
- **Tab**: Exit radio group

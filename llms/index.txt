# BlazorUI Component Library - LLM Coding Agent Guide

**Project:** BlazorUI - A comprehensive Blazor component library based on shadcn/ui design
**Repository:** https://github.com/blazorui-net/ui

---

## Package Overview

### BlazorUI.Primitives
**Purpose:** Headless, unstyled components providing behavior and accessibility without styling
**Use when:** You need full control over styling and appearance

### BlazorUI.Components
**Purpose:** Pre-styled components with shadcn/ui design using Tailwind CSS
**Use when:** You want beautiful, ready-to-use components with minimal configuration

### BlazorUI.Icons
**Purpose:** Lucide icon library integration with 1,640+ icons
**Use when:** You need scalable, customizable SVG icons

## Key Architecture Patterns

### 1. Two-Layer Architecture
**Primitives Layer:** Headless components with behavior and accessibility, zero styling
**Components Layer:** Pre-styled components built on primitives with shadcn/ui design

**When to use Primitives:** Need full control over styling
**When to use Components:** Want shadcn/ui design out-of-the-box

### 2. Component Naming Convention
**Primitives:** Add "Primitive" suffix when importing
**Components:** No suffix needed

### 3. State Management Patterns

**Uncontrolled Mode:** Component manages its own state
```razor
<Tabs DefaultValue="tab1">
    <TabsList>...</TabsList>
</Tabs>
```

**Controlled Mode:** Parent manages state with @bind-
```razor
<Tabs @bind-Value="currentTab">
    <TabsList>...</TabsList>
</Tabs>
```

### 4. Composition Pattern

Components use `ChildContent` for flexible composition:
```razor
<Card>
    <CardHeader>
        <CardTitle>Title</CardTitle>
    </CardHeader>
    <CardContent>Content</CardContent>
</Card>
```

### 5. Portal Pattern

Overlay components (Dialog, Sheet, Popover, Tooltip, HoverCard, Select, Combobox) use portals for proper z-index stacking.

### 6. Styling System

**CSS Variables:** Theme system using HSL color values
**Tailwind CSS:** Utility-first styling
**Dark mode:** Built-in light/dark theme support

Build a new Blazor UI Component following the structure already in use with existing components.

Use the DevFlow:build-feature command to manage the implementation of this component.

Example: /DevFlow:build-feature $ARGUMENTS

**Important, when running the /DevFlow:build-feature command, include the following additional instructions**

- Don't include a testing task in the implementation task list
- Always make sure the task list it kept up-to-date as tasks are completed
- Always create a demo page for the new component
- Always add the demo page to the Components homepage and the Components menu
- Always reference any image provided
- Always reference any source examples or links to source code provided
